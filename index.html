Zoom in and click on any city you want on the map above to change its color cyclically.
All the information used for the drawing can be edited in the text box below (in JSON format). Copy it to a convenient location to continue next time. Click the blank area to refresh the data.
Modify the variable color_list and run refresh_map() in the browser console to change the color to be rendered.
